# Category Icons

This directory contains SVG icon files for the category filters in the sidebar.

## Icon Files

- `all-types.svg` - Icon for "All Types" filter
- `outdoor.svg` - Icon for Outdoor category (green #10b981)
- `taste.svg` - Icon for Taste category (orange #f59e0b)
- `stay.svg` - Icon for Stay category (blue #3b82f6)
- `culture.svg` - Icon for Culture category (purple #8b5cf6)
- `experience.svg` - Icon for Experience category (red #ef4444)
- `community.svg` - Icon for Community category (cyan #06b6d4)

## Editing Icons

You can edit these SVG files directly. The icons are loaded dynamically from these files, so any changes you make will be reflected immediately when you refresh the page.

### Color Guidelines

Each icon uses a specific color defined in the SVG:
- Outdoor: `#10b981` (green)
- Taste: `#f59e0b` (orange/amber)
- Stay: `#3b82f6` (blue)
- Culture: `#8b5cf6` (purple)
- Experience: `#ef4444` (red)
- Community: `#06b6d4` (cyan)

You can change these colors by editing the `stroke` or `fill` attributes in the SVG files.

### Size

Icons are displayed at 24x24 pixels by default. The `width` and `height` attributes in the SVG files control the base size, but they will be scaled to fit the container.
